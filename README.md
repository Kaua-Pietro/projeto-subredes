# 🌐 Calculadora de Sub-redes IPv4

Uma aplicação web desenvolvida em **React, TypeScript e Vite** para auxiliar no estudo de **Redes de Computadores**, permitindo realizar cálculos de subnetting de forma visual, prática e interativa.

## 🚀 Funcionalidades

* Cálculo de máscara de sub-rede
* Identificação do endereço de rede
* Identificação do endereço de broadcast
* Cálculo do primeiro e último host válido
* Quantidade de hosts disponíveis
* Conversão de IP para binário
* Demonstração da operação lógica AND
* Exibição da divisão entre bits de rede e host
* Geração de tabelas de sub-redes
* Simulação visual de VLSM
* Identificação da classe do IP
* Identificação do tipo de IP (Público ou Privado)

## 🖥️ Tecnologias Utilizadas

* React
* TypeScript
* Vite
* CSS

## 🎯 Objetivo

O objetivo deste projeto é facilitar o aprendizado dos conceitos de endereçamento IPv4, CIDR, subnetting e VLSM através de uma interface visual e intuitiva.

## 📚 Conceitos Abordados

* Endereçamento IPv4
* Máscaras de Sub-rede
* CIDR
* Subnetting
* VLSM (Variable Length Subnet Mask)
* Operação AND
* Conversão Binária
* Classes de IP
* Redes Públicas e Privadas
